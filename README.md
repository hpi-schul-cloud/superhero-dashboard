# SuperHero-Dashboard

## Requirements

* Node.js must be installed in the version specified in package.json.
* For the superhero-dashboard to work the schulcloud-server must be running. Furthermore, in order to use the batch deletion feature, the admin-api-server must also be running. See the [schulcloud-server repository](https://github.com/schulcloud/schulcloud-server) for how to run them.

## Setup

1. Clone the repository.
2. Go into the cloned folder and run `npm install`.

## Run

1. Run `npm run build` to build static files.
2. Run `npm run watch` (or another start script from package.json) to start the application.
